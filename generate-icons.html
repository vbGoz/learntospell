Instructions:

  1. Right-click on each canvas above
  2. Select "Save Image As..." or "Download Image"
  3. Save the 192x192 canvas as icon-192.png
  4. Save the 512x512 canvas as icon-512.png
  5. Place both files in the root directory of your project (next to index.html)

Alternative: Use a Design Tool

  • Figma (free): Create a 512x512 square with purple gradient and "W" text
  • Canva (free): Use icon templates
  • Online Icon Generator: PWA Builder Image Generator

Icon Requirements:

  • 192x192 pixels: Required for PWA
  • 512x512 pixels: Required for splash screens
  • Format: PNG with transparency
  • Colors: Purple theme (#9333ea) to match app
